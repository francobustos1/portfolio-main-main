# Portfolio Profesional - Franco Bustos

Este repositorio contiene el código fuente de mi **portfolio personal** como Técnico en Programación de Videojuegos.  
El objetivo del sitio es presentar mi perfil profesional, mostrar proyectos y ofrecer canales de contacto para clientes y equipos interesados.

---

## Estructura del repositorio

- `index.html` — Página principal / Portada (con foto, proyecto destacado y enlaces).
- `servicios.html` — Página de servicios (lo que ofrezco: desarrollo de videojuegos, reparación de PC, programación, narrativa).
- `CV.html` — Curriculum Vitae (datos personales, perfil, educación, habilidades, proyectos).
- `sobre-mi.html` — Sobre mí (biografía y objetivos).
- `contacto.html` — Contacto (formulario funcional que abre el cliente de correo).
- `styles.css` — Estilos globales, responsive y diseño.
- `script.js` — Funcionalidad: menú hamburguesa, año automático en index y manejo del formulario.
- `foto.jpg` — Foto de perfil (colocar en la raíz del proyecto; nombre exacto `foto.jpg`).
